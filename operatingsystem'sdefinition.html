Operating System's definition:

An operating system is a software that communicates with the hardware and allows other programs to run and it is the most important software that runs on a computer it manages the computer's memory and processes as well as all of its software and hardware it also allows you to communicate with the computer without knowing how to speak the computer's language without an operating system a computer is useless
