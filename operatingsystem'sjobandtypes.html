Operating system's job and types:

Operating system manages all of the software and hardware on the computer most of the time there are several computer programs running at the same time and they all need to access your computer's central processing unit (CPU) memory and storage the operating system coordinates all of this to make sure each program gets what it needs and the types is Batch operating system, Multitasking/time sharing, Mltiprocessing OS, Real Time OS, Distributed OS, Network OS and Mobile OS
